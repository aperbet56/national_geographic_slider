## NATIONAL GEOGRAPHIC SLIDER

## Le challenge

Création d'un carrousel d'images élégant et interactif avec lecture automatique et lecture manuelle en HTML5, CSS3 et JavaScript.

## Démonstration

Lien vers le projet : https://aperbet56.github.io/national_geographic_slider/

## Développez avec

- Utilisation des balises sémantiques HTML5
- CSS3
- Flexbox
- Animations CSS
- Desktop first
- Page web responsive
- Importation de la police "Poppins"
- Utilisation d'un normaliseur : le fichier normalize.css
- JavaScript
- Code JavaScript commenté
